# Dra. Vanessa Costa

Landing page estática para estética premium em Manaus, focada em conversão via WhatsApp.

## Estrutura

- `index.html`
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/img/`
- `netlify.toml`
- `AGENTS.md`

## Deploy

O projeto não usa build, dependências NPM ou framework. No Netlify, publique a raiz do repositório.

```toml
[build]
  publish = "."
  command = ""
```
